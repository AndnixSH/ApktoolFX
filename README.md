# ApktoolFX
A GUI for Apktool to make reverse engineering of android apps a breeze.

ApktoolFX is a GUI for [Apktool](https://github.com/iBotPeaches/Apktool) written in JavaFX

Features:
  - Drag and drop
  - Material design
  - Log (with color-coding)
